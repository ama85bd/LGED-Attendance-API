## To create new solution file inside main folder (run bellow cmd in terminal)
-dotnet new sln

## To create webapi go to inside that folder
-dotnet new webapi -n LGED.Web

## To create class library go to inside that folder
-dotnet new classlib -n LGED.Core

## Add this project into solution
-dotnet sln add Web/LGED.Web 

## Add dependency one folder to another folder
-dotnet add reference ../../Core/LGED.Core

## for db context migration ## 
  ### for creating migrations
-dotnet ef migrations add InitDatabase --project Model\LGED.Model\ -s Web\LGED.Web\ -c LgedDbContext --verbose

  ### for updating db
-dotnet ef database update InitDatabase --project Model\LGED.Model\ -s Web\LGED.Web\ -c LgedDbContext --verbose 

  ### for removing db
-dotnet ef migrations remove --project YourDataAccessLibraryName -s YourWebProjectName -c YourDbContextClassName --verbose
